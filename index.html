Eksempel på installasjon av Pillow med pip

Dette er en liten guide for hvordan man kan installere Pillow ved hjelp av pip.


Stegene:

1. Åpne terminal eller kommandolinje

  • Windows: Åpne "cmd" eller "PowerShell"
  • macOS/Linux: Åpne "Terminal"

2. Sjekk at du har pip installert

Dette er ikke nødvendig, men det kan være nyttig å vise elevene.

pip --version

Hvis du vil være sikker på at du bruker riktig pip for Python 3, kan du skrive:

python3 -m pip --version

3. Installer Pillow med pip

For å installere Pillow skriver du:

pip install Pillow

Hvis du vil forsikre deg om at du bruker pip knyttet til Python 3, kan du gjøre:

python3 -m pip install Pillow

4. Verifiser installasjonen

Etter installasjonen kan du liste opp installerte pakker for å sjekke at Pillow er med:

pip list
